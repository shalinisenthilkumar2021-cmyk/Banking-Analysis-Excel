# 🏦 Banking Data Analysis Dashboard using Excel & SQL

## 📌 Project Overview

The Banking Data Analysis Dashboard is an interactive data analytics project developed using Microsoft Excel and SQL. The project analyzes customer, transaction, loan, and branch data to generate meaningful business insights through dashboards, pivot tables, charts, and SQL queries.

This project demonstrates skills in data cleaning, data visualization, dashboard creation, business intelligence, and SQL-based data analysis.

---

## 🎯 Objectives

- Analyze customer demographics and banking behavior.
- Monitor transaction trends and spending patterns.
- Analyze loan approval and loan distribution.
- Compare branch performance.
- Create an interactive dashboard for business decision-making.
